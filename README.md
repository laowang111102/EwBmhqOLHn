# 家政管理系统 python663

家政管理系统是一款基于Python Django框架和MySQL数据库开发的应用，旨在帮助家政服务公司高效管理业务和客户信息。

## 技术栈

- **后端**: Python Django
- **数据库**: MySQL
- **前端**: HTML, CSS, JavaScript

## 功能模块

- **用户管理**: 管理员可以添加、删除、编辑用户信息。
- **服务项目管理**: 对各项家政服务进行分类和详细描述。
- **预约管理**: 客户可以在线预约服务，管理员可查看和调整预约。
- **财务管理**: 管理服务收费和财务报表。
- **评价管理**: 客户可以对服务进行评价，管理员可查看和回复评价。

## 系统角色

- **管理员**: 管理整个系统，包括用户、服务、预约等。
- **普通用户**: 可以查看服务、预约服务并给予评价。

## 运行环境

- Python 3.8 或以上版本
- Django 3.x
- MySQL 5.7 或以上版本

## 部署步骤

1. 安装Python和MySQL环境。
2. 使用pip安装Django和相关的依赖包。
3. 在MySQL中创建数据库并配置Django以连接该数据库。
4. 运行Django项目进行迁移数据库表结构。
5. 启动项目服务。
